# 📖 Recipe Voice App - פרויקט סיכום למידת AI

**פרויקט זה הוגש כתרגיל סיכום במסגרת קורס למידת AI.** הפרויקט מדגים שילוב בין פיתוח Web לבין יכולות אינטראקטיביות מתקדמות, תוך דגש על ייעול הקוד ושימוש מושכל בכלי בינה מלאכותית לתכנון ובניית המערכת.

## 🔗 קישורים לפרויקט
* **קישור לאתר ב-Netlify (חי):** [cook-together.netlify.app](https://cook-together.netlify.app/)
* **קישור לאתר ב-GitHub Pages:** [tamarsherman.github.io/recipe-voice-app](https://tamarsherman.github.io/recipe-voice-app/)

---

## 🛠 תכונות האתר (Features)

### 1. מערכת כניסה והגדרות אישיות
* **מסך Login:** כניסה מאובטחת המבוססת על נתונים שנשמרים ב-Local Storage(דוגמא למשתמש:admin סיסמה:1234).
* **התאמה אישית:** ממשק המאפשר למשתמש לקבוע את קצב הקראת המתכון (זמן המתנה בין שורות) ולבחור בין מצב תצוגה כהה (Dark Mode) לבהיר.

### 2. ניהול מתכונים חכם
* **רשימת מתכונים:** תצוגה דינמית של כל המתכונים במערכת.
* **חיפוש והוספה:** אפשרות לחיפוש מהיר של מתכון קיים או הוספת מתכון חדש הכולל מצרכים ושלבי הכנה מפורטים.

### 3. הקראה קולית (Speech Synthesis)
* **עוזר אישי במטבח:** האתר מקריא את שלבי המתכון בקול כדי לאפשר עבודה רציפה במטבח ללא צורך במגע במסך.
* **שליטה מלאה:** כפתורי הפעלה ועצירה לשליטה בתהליך ההקראה.

---

## 🤖 דגשים מפרויקט למידת AI
במסגרת תרגיל הסיכום, הפרויקט בוצע תוך יישום עקרונות שנלמדו בקורס:
* **שימוש בכלי AI:** הסתייעות בבינה מלאכותית לכתיבת קוד יעיל (Optimization), ביצוע Code Review ומציאת מקרי קצה.
* **ייעול הקוד:** כתיבת לוגיקת JavaScript נקייה, מודולרית ומובנת.
* **חווית משתמש (UX):** פתרון בעיה ממשית באמצעות טכנולוגיה (בישול חופשי ללא מגע).

---

## 💻 טכנולוגיות בשימוש
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Storage:** Browser Local Storage.
* **Deployment:** Netlify & GitHub Pages.
* **AI Tools:** שימוש ב-AI לשיפור וייעול תהליך הפיתוח.
