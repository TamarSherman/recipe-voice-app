📖 Recipe Voice App - פרויקט סיכום למידת Ai
פרויקט זה הוגש כתרגיל סיכום במסגרת קורס למידת AI. הפרויקט מדגים שילוב בין פיתוח Web לבין יכולות אינטראקטיביות, בדגש על ייעול הקוד ושימוש מושכל בכלי עזר טכנולוגיים.

🔗 קישורים לפרויקט
קישור לאתר ב-Netlify (חי): cook-together.netlify.app

קישור לאתר ב-GitHub Pages: tamarsherman.github.io/recipe-voice-app

🛠 תכונות האתר (Features)
1. מערכת כניסה והגדרות אישיות
מסך Login מאובטח מול נתונים ב-Local Storage.

ממשק הגדרות משתמש המאפשר התאמה אישית של זמן ההקראה ותצוגת האתר (מצב בהיר/כהה).

2. ניהול מתכונים חכם
רשימת מתכונים דינמית עם אפשרות לחיפוש מהיר.

טופס להוספת מתכונים חדשים הכולל חומרים ושלבי הכנה.

3. הקראה קולית (Voice Recognition & Synthesis)
מסך מתכון ייעודי המאפשר הקראה אוטומטית של שלבי ההכנה.

שליטה מלאה למשתמש: כפתורי עצירה, המשך והתאמת קצב הדיבור.

🤖 דגשים מפרויקט למידת AI
במסגרת הפרויקט הושם דגש על:

ייעול קוד: כתיבת פונקציות JavaScript נקיות וחסכוניות.

חווית משתמש (UX): יצירת ממשק נוח לעבודה במטבח "בידיים מלאות" בעזרת הקול.

פתרון בעיות: התמודדות עם מקרי קצה בניהול נתונים מקומי (Local Storage).

💻 טכנולוגיות בשימוש
Frontend: HTML5, CSS3, JavaScript (ES6+).

Storage: Browser Local Storage.

Deployment: Netlify & GitHub Pages.

Tools: AI assisted development & Code Review.

הוגש ע"י: תמר שרמן במסגרת מטלת סיכום - קורס AI.
