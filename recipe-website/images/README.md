# תיקיית תמונות למתכונים

## הוראות:
שים כאן תמונות למתכונים שלך.

## תמונות נדרשות:
1. **pasta.jpg** - תמונה של תמונותפסטה ברוטב עגבניות
2. **salad.jpg** - תמונה של סלט ירקות
3. **cake.jpg** - תמונה של עוגת שוקולד

## איך להוסיף תמונות:
1. הורד  מהאינטרנט או השתמש בתמונות שלך
2. שמור אותן בתיקייה הזו עם השמות המתאימים
3. התמונות יופיעו אוטומטית באתר

## פורמטים נתמכים:
- JPG/JPEG
- PNG
- GIF
- WebP

## גודל מומלץ:
- רוחב: 400-800 פיקסלים
- גובה: 300-600 פיקסלים
